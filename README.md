# 個人用ライフプランシミュレーション（HTML/JS版） 開発仕様書

## 1. 概要
Mac環境でローカルに自己完結して動作するライフプランシミュレーションツール。  
ブラウザ上でフォーム入力、計算、グラフ表示、PDF/Excel出力が可能。  
将来の収支・資産推移、教育費・住宅費ピーク時期、保険不足額を可視化。

---

## 2. ターゲットユーザー
- 個人・家族向け
- 将来の資産計画や保険必要額を把握したいユーザー
- Mac環境でExcelなしでも利用可能

---

## 3. 機能要件

### 3.1 入力フォーム
**基本情報**
- 本人・配偶者
  - 年齢、性別、年収（手取り）、給与増加率、昇給停止年齢、定年（退職）年齢、嘱託年収
- 家族構成
  - 子ども人数、年齢、出生予定年
- 現在の資産
  - 現金、預金、投資、年金見込み

**支出情報**
- 生活費（月/年）
- 住宅費（ローン残高、返済期間、金利、家賃）
- 教育費（子どもごとに設定、幼稚園〜大学＋習い事）
- 医療費・保険料
- その他支出（車・交通・娯楽費など）

**保険・リスク情報**
- 生命保険（死亡保障、掛け金、期間）
- 医療保険（入院日額、通院保障）
- シナリオ設定（インフレ率、投資利回り変動）

---

### 3.2 シミュレーション結果
- 年ごとの収支・貯蓄・資産推移（折れ線グラフ）
- 支出構成比（円グラフ）
- 教育費・住宅費ピーク時期の可視化
- 保険で補えるリスクと不足額表示
- シナリオ比較（インフレ率や投資利回りの変化）

---

### 3.3 出力機能
- PDF出力（グラフ＋表＋前提条件まとめ、jsPDF + html2canvas使用）
- Excel出力（年ごとの詳細シミュレーション表、SheetJS使用）
- データ保存：ブラウザローカルストレージまたはファイル書き出しで対応

---

## 4. 計算式（基本例）

**収支**

年間収入 = 本人年収 + 配偶者年収（あれば）
※ 収入は独自の「4段階収入モデル」（昇給期 → 横ばい期 → 嘱託期 → 年金期）を用いて推移を計算します。
※ 住宅ローン控除が適用される期間は、還付金として年間収入に加算されます。

年間支出 = 生活費 + 住宅費 + 教育費 + 医療費 + その他支出
年間貯蓄増減 = 年間収入 - 年間支出

---

## 資産推移
資産[t+1] = 資産[t] + 年間貯蓄増減 + 投資リターン（資産[t] × 投資利回り）

## 教育費
子ども教育費[t] = 年齢別教育費 × (1 + インフレ率)^(t - 現在年)

## 住宅ローン返済
年間住宅ローン = 残高 × (金利/12) × (1+金利/12)^返済月数 / ((1+金利/12)^返済月数 - 1) × 12

## 保険必要額
必要保障額 = 将来不足資金 - 現在資産 - 既存保険保障


--- 

## 5. 技術スタック案
HTML / CSS / JavaScript（ブラウザだけで完結）
UIライブラリ：TailwindCSS（オプション）
グラフ描画：Chart.js または Recharts
PDF出力：jsPDF + html2canvas
Excel出力：SheetJS (xlsx)
データ保存：localStorage または JSONファイル書き出し
OS依存なし、MacでもWindowsでもブラウザで動作可能

## 6. UI/UXポイント
入力フォームは段階的に分けて操作しやすく
子どもごとの教育費や出生予定を簡単に追加可能
シナリオ比較はスライダーやドロップダウンで切替可能
出力ボタン1つでPDF・Excel生成
ローカル完結型で個人情報や資産データが外部に出ない設計

## 7. 補足
個人利用のためUIはシンプルかつ直感的に
将来的にシナリオ保存や再利用も可能にすると便利
開発難易度を抑えたい場合、まずはHTML + JS + Chart.jsでプロトタイプ作成後、PDF/Excel出力を追加すると効率的
